# Тестовый проект mssql

## Запуск проекта

```bash
sudo docker compose up
sudo docker compose up -d
```

## Остановка проекта

```bash
sudo docker compose stop
sudo docker compose down
sudo docker compose down -v
```

## Посмотреть лог

```bash
sudo docker logs erm
```

## Рестарт

```bash
sudo docker-compose restart erm
```
